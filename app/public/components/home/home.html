Angular Seed

By Ian Civgin

How to use this seed:

  • 1. Fork and clone this repo.
  • 2. Run npm install
  • 3. Start your server with nodemon or "npm start"
  • 4. Change app name in all app.js, index.html, controllers.js, and services.js
  • 4. Modify to your liking :)
